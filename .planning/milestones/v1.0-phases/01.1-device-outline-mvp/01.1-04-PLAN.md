<objective>
Fix the slug mismatch that prevents 9 of 10 priority devices from rendering SVG outlines, and fix the export cache timing gap that prevents outlinePath from appearing in exported JSON.

Purpose: The outline pipeline (generation CLI, SVG assets, loading module, renderer) is fully built and wired, but a key namespace split between DEVICES constant keys (short form like `usw-lite-16`) and outline index slugs (full form like `usw-lite-16-poe`) prevents outlines from loading for all devices except `ux7`. Additionally, FrontView only loads front-face outlines but export reads top-face from cache, which is never populated.

Output: All 10 priority devices render with outlines in FrontView; export JSON includes outlinePath for devices with top-face outlines.
</objective>

   To load both faces:
   ```typescript
   const promises: Promise<string | null>[] = [];

     if (hasOutline(el.key, 'top') && getCachedOutlinePath(el.key, 'top') === null) {
   ```
