## Decisions Made
- Extracted shared SVG path utilities into `scripts/lib/svg-path-utils.ts` rather than keeping them inline in generate-outline.ts -- avoids circular dependency with validate-outlines.ts
- Scripts tsconfig.json uses independent configuration (NodeNext) rather than extending root tsconfig (bundler) -- different module resolution needs
- Arc bounding box uses conservative endpoint +/- radius approach -- exact arc math would be more precise but unnecessary for simple device outlines
