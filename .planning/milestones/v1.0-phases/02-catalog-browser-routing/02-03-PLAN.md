2. `extractSerializable(state: ConfigState): SerializedDesign` — extracts the serializable subset from full store state. Add `v: 1` version tag.

3. `encodeDesign(design: SerializedDesign): string` — JSON.stringify, TextEncoder for UTF-8 safety, btoa for base64. Per CONTEXT.md: no compression, plain base64.
   ```typescript
   export function encodeDesign(design: SerializedDesign): string {
     const json = JSON.stringify(design);
     const bytes = new TextEncoder().encode(json);
     const binary = String.fromCharCode(...bytes);
     return btoa(binary);
   }
   ```

4. `decodeDesign(encoded: string): SerializedDesign | null` — atob, TextDecoder, JSON.parse, validate `v === 1`. Return null on any failure (malformed base64, invalid JSON, wrong version).
   ```typescript
   export function decodeDesign(encoded: string): SerializedDesign | null {
     try {
       const binary = atob(encoded);
       const bytes = Uint8Array.from(binary, c => c.charCodeAt(0));
       const json = new TextDecoder().decode(bytes);
       const parsed = JSON.parse(json);
       if (parsed.v !== 1) return null;
       return parsed as SerializedDesign;
     } catch {
       return null;
     }
   }
   ```

5. `applyDesignToStore(design: SerializedDesign): void` — calls `useConfigStore.setState()` with all fields from the serialized design, excluding `v`.

   IMPORTANT per CONTEXT.md locked decision: When a shared URL references a device slug not in the current catalog, load the element using saved dimensions from URL state. The elements array in SerializedDesign includes `w` and `h`, so the element will render with those dimensions even if the device slug is not recognized. Do NOT filter out unknown slugs.

**Create `src/hooks/useDesignPersistence.ts`:**

This hook handles two concerns: (A) localStorage auto-save, and (B) initial state load from URL or localStorage.

**Part A: localStorage auto-save (debounced 500ms per CONTEXT.md):**

1. `initDesignPersistence()` — non-hook function that subscribes to `useConfigStore` outside React:
   ```typescript
   const STORAGE_KEY = 'rackpro-design';
   const DEBOUNCE_MS = 500;

   export function initDesignPersistence(): () => void {
     let timer: ReturnType<typeof setTimeout> | null = null;

     const unsubscribe = useConfigStore.subscribe((state) => {
       if (timer) clearTimeout(timer);
       timer = setTimeout(() => {
         try {
           const serializable = extractSerializable(state);
           localStorage.setItem(STORAGE_KEY, JSON.stringify(serializable));
         } catch {
           // Quota exceeded — silent fail (per RESEARCH.md pitfall 4)
         }
       }, DEBOUNCE_MS);
     });

     return unsubscribe;
   }
   ```

2. `restoreFromLocalStorage(): boolean` — reads STORAGE_KEY, parses JSON, applies to store via `applyDesignToStore`. Returns true if restored successfully.

**Part B: Initial load logic (URL wins per CONTEXT.md):**

3. `loadInitialDesign()` — called once on mount:
   - Parse the URL hash for a `design` search param. With TanStack Router hash history, the URL looks like `#/?design=...` or `#/catalog?design=...`. Parse using `window.location.hash`:
     ```typescript
     const hashContent = window.location.hash.slice(1); // remove leading #
     const queryIndex = hashContent.indexOf('?');
     const searchStr = queryIndex >= 0 ? hashContent.slice(queryIndex) : '';
     const params = new URLSearchParams(searchStr);
     const designParam = params.get('design');
     ```
   - If `designParam` exists and `decodeDesign(designParam)` succeeds:
     - Call `applyDesignToStore(design)`
     - Show toast: `showToast('Loaded shared design. Your saved design is still available.', { label: 'Restore saved', onClick: () => restoreFromLocalStorage() })`
     - Clean up: remove `design` param from URL to prevent re-loading on refresh. Use `window.history.replaceState` to strip the param without triggering navigation.
     - Return (skip localStorage load)
   - If no URL design param: call `restoreFromLocalStorage()`

**Part C: Share URL generator:**

4. `generateShareUrl(): string` — reads current store state, extracts serializable, encodes to base64, constructs URL:
   ```typescript
   export function generateShareUrl(): string {
     const state = useConfigStore.getState();
     const design = extractSerializable(state);
     const encoded = encodeDesign(design);
     const base = window.location.href.split('#')[0];
     return `${base}#/?design=${encoded}`;
   }
   ```

**Part D: React hook wrapper:**

5. `useDesignPersistence()` — React hook called in root layout:
   ```typescript
   export function useDesignPersistence() {
     useEffect(() => {
       // Load initial design (URL or localStorage)
       loadInitialDesign();

       // Start auto-save subscription
       const unsubscribe = initDesignPersistence();
       return unsubscribe;
     }, []);
   }
   ```
