# Plan 08-02: Layout Components — Header, Sidebar, Nav, Status

## Goal
Restyle the four layout shell components (Header, NavSidebar, Sidebar, StatusBar) and MainContent for the light theme. Fix tiny text sizes, dense spacing, and teal accent references. After this plan, the app chrome looks professional and readable.

## Must-Haves

### Truths
- Header: clean white/light gray background, orange accent for active states
- NavSidebar: light background, orange selected indicator, readable labels (min 12px)
- Sidebar: all `text-[7-11px]` replaced with `text-xs`/`text-sm`, spacing loosened
- StatusBar: light background, readable text
- MainContent: proper padding/gaps
- No monospace font on labels/headings (only on values/code)
- All hardcoded teal/cyan colors replaced

- `src/components/Header.tsx`
- `src/components/NavSidebar.tsx`
- `src/components/Sidebar.tsx`
- `src/components/StatusBar.tsx`
- `src/components/MainContent.tsx`

## Key Links
- `08-CONTEXT.md` — Design decisions

## Execution

### Step 1: Header.tsx
- Background: `bg-card` (white) with subtle bottom border
- Text: `text-foreground` (dark), logo/brand in orange
- Remove any teal/cyan references
- Ensure min text size 14px

### Step 2: NavSidebar.tsx
- Background: `bg-card` (white) with right border
- Nav items: `text-muted-foreground` default, `text-primary` when active with orange left-edge indicator
- Icons: sized appropriately, `text-muted-foreground`
- Remove all tiny text, ensure `text-sm` minimum

### Step 3: Sidebar.tsx (heaviest — 41 tiny text occurrences, 18 dense spacing)
- All `text-[7px]` through `text-[11px]` → `text-xs` (12px) or `text-sm` (14px)
- All `gap-[5px]` / `gap-1` → `gap-2` (8px) minimum
- All `py-1` / `p-1` → `py-2` / `p-2` minimum
- Section headers: `text-sm font-semibold text-foreground`
- Values: keep monospace via `font-mono` class
- Labels: sans-serif (inherits Inter)
- Background: `bg-background` (near-white) or `bg-card` (white)

### Step 4: StatusBar.tsx
- Background: `bg-card` (white) with top border
- Text: `text-muted-foreground`, min `text-xs`
- Status indicators: orange for active, green for success

### Step 5: MainContent.tsx
- Padding: `p-4` minimum
- Gaps: `gap-4`
- Background: `bg-background`

### Step 6: Verify build
Run `npm run build` and fix any breakage.
