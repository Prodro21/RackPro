<objective>
Audit all Zustand selectors returning objects/arrays, add module-level memoization where missing, document cache key composition, and write stability tests.

Purpose: React 19's strict `useSyncExternalStore` tear detection causes infinite re-render loops when selectors return new references on every call. This plan ensures every selector is memoized and tested for referential stability, preventing subtle performance bugs and render loops.

Output: All selectors memoized with documented cache keys. A stability test suite verifying `===` equality on repeated calls.
</objective>
