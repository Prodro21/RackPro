### Integration Wiring: 21/22 Connected

All 22 phase exports are consumed downstream. 3 partial wiring gaps found:

1. **`configJson.ts` line 69** — Uses `CONNECTORS[e.key]` instead of `lookupConnector(e.key)`. Catalog-only connectors will have missing cutout data in exported JSON.
2. **`selectors.ts` line 168** — `selectMaxDeviceDepth` uses `CONNECTORS[el.key]` for connector depth. Catalog-only connectors resolve to 0mm, causing incorrect enclosure depth.
3. **`SideView.tsx` line 131** — Same pattern. Side profile connector depth indicators will be wrong for catalog-only slugs.

**Impact:** Currently low — all tested connector slugs exist in both inline constants and catalog JSON. Will become a real bug as catalog grows with entries not in `src/constants/connectors.ts`.
