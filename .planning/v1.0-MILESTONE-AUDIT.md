## Critical Gap: Export Pipeline Connector Lookup

**File:** `src/export/configJson.ts` line 69
**Code:** `const con = e.type === 'connector' ? CONNECTORS[e.key] : undefined;`

The inline `CONNECTORS` constant contains 10 connector keys. The catalog store contains 37. The 27 catalog-only slugs (`keystone`, `bnc-flange`, `n-type`, `tnc`, `lc-duplex`, `sc-simplex`, `lgx-fiber-panel`, `de-9`, `da-15`, `db-25`, `usb-b`, `usb-c`, `displayport`, `iec-c14`, `iec-c13`, `powercon-20a`, `powercon-true1`, `xlr3-chassis`, `xlr5-chassis`, `speakon-chassis`, `banana-post`, `rca-single`, `rca-dual`, `trs-35mm`, `trs-635mm`, `din-5-midi`, `toslink`) resolve to `undefined`, producing `cutout: 'rect'` and `depthBehind: 0` in all 4 export formats.

`validation.ts` correctly uses `lookupConnector()` for MISSING_DEF checking, so these connectors pass preflight — exports proceed with silently wrong geometry.

**Fix:** Replace `CONNECTORS[e.key]` with `lookupConnector(e.key)` and add the import.

**Affected requirements:** CAT-02, EXP-01, EXP-04
**Affected flow:** Catalog connector → Add to Panel → Export (wrong cutout geometry)

## Minor Gap: SpecsTab Reactive Badge
