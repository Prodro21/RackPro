### Integration Issues (2)

**1. selectMaxDeviceDepth connector lookup (functional)**

`src/store/selectors.ts` line 182 uses `CONNECTORS[el.key]?.depthBehind` for connector elements instead of `lookupConnector(el.key)?.depthBehind`. Catalog-sourced connectors not duplicated in inline constants contribute 0mm to enclosure depth.

**Fix:** Replace with `(lookupConnector(el.key)?.depthBehind ?? 0)`.

**2. ExportTab preflight staleness (display-only)**

`src/components/ExportTab.tsx` line 116 keys preflight effect on `elements.length` instead of element positions. Displayed `PreflightReport` can show stale results after drags. Export blocking still works correctly via `checkBeforeExport()` at click time.

**Fix:** Add `positionKey` dependency matching `StepReview.tsx` pattern.
